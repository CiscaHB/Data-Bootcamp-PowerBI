# Data-Bootcamp-PowerBI: Retail & Sales Analysis Project

## 📌 Overview
This project was completed as part of a **Data Technician Bootcamp**, focusing on building end-to-end data analytics solutions using **Power BI**. The project transforms raw retail and sales data into meaningful insights through effective data modelling, reporting, and visual storytelling.

It demonstrates the ability to clean, analyse, and visualise data to support business decision-making.

---

## 🛠️ Key Skills & Tools

### 🔄 Data Transformation & Cleaning
- Used **Power Query** to:
  - Clean and standardise raw datasets
  - Handle missing values and inconsistencies  
  - Merge and transform multiple data sources  
- Implemented best practices for preparing data for analysis  

### 📐 Data Modelling & DAX
- Created **calculated columns** and **measures** using **DAX (Data Analysis Expressions)**  
- Developed business logic for KPIs such as:
  - Total Sales  
  - Profit Margins  
  - Year-over-Year trends  
- Built a structured data model to support reporting and analysis  

---

## 📊 Data Visualisation & Reporting

### 📈 Interactive Dashboards
- Designed dynamic Power BI reports using:
  - **Slicers** for filtering by date, category, and region  
  - **Filters** to drill down into specific segments  
- Enabled user-driven exploration of retail performance data  

### 📉 Visualisations Created
- **Bar Charts** – compare sales across categories and regions  
- **Line Charts** – analyse trends over time  
- **Pie Charts** – visualise proportional breakdowns  
- **Maps** – display geographic distribution of sales  

---

## 📖 Data Storytelling
- Translated complex retail datasets into **clear, actionable insights**  
- Focused on storytelling techniques to:
  - Highlight key performance trends  
  - Identify high-performing products and regions  
  - Reveal patterns in customer purchasing behaviour  
- Designed visuals to guide stakeholders through a logical analytical narrative  

---

## 🎯 Project Outcomes

  - Insight into retail sales performance  
  - Easy exploration through filters and drill-downs  
  - Clear communication of trends and patterns
